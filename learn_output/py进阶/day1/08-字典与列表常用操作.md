# 🐍 08 — 字典与列表常用操作

## 本项目用到的字典操作

### 1. 字典的 get() 方法 — 安全取值

```python
# tools/get_attraction.py
response = tavily.search(...)

# ❌ 可能报错：如果 response 里没有 "answer" 这个键
result = response["answer"]

# ✅ 安全：没有 "answer" 就返回 None
result = response.get("answer")

# ✅ 更安全：没有就返回默认值
response.get("results", [])  # 没有 results 字段就返回空列表
```

### 2. 字典的键值对遍历

```python
# agent_core/parser.py — 把函数参数解析成字典
kwargs = dict(re.findall(r'(\w+)="([^"]*)"', args_str))
# 结果: {"city": "北京"} 或 {"city": "北京", "weather": "晴"}
```

### 3. 用 `**kwargs` 把字典展开为参数

```python
# agent_core/agent.py
kwargs = {"city": "北京"}
available_tools["get_weather"](**kwargs)
# 等价于：available_tools["get_weather"](city="北京")
```

## 本项目用到的列表操作

### 1. 列表推导式

```python
# tools/get_attraction.py
formatted_results = []
for result in response.get("results", []):
    formatted_results.append(f"- {result['title']}: {result['content']}")
```

用列表推导式可以写得更简洁（**进阶**写法，理解即可）：

```python
formatted_results = [
    f"- {r['title']}: {r['content']}"
    for r in response.get("results", [])
]
```

### 2. 列表拼接为字符串

```python
"\n".join(formatted_results)
# 把列表用换行符连接成一个字符串
```

### 3. 用字典模拟 switch-case

```python
# tools/__init__.py — 把工具按名称存到字典
available_tools = {
    "get_weather": get_weather,       # get_weather 是函数
    "get_attraction": get_attraction,
}

# agent_core/agent.py — 按名称调用
if action_type in available_tools:
    observation = available_tools[action_type](**kwargs)
# 等价于：if action_type == "get_weather": get_weather(**kwargs)
#         elif action_type == "get_attraction": get_attraction(**kwargs)
```

## 高频操作速查

### 列表

```python
lst = [1, 2, 3]

lst.append(4)                    # 添加 → [1, 2, 3, 4]
lst.pop()                        # 移除最后一个 → 返回 4
lst[0]                           # 取第一个 → 1
lst[-1]                          # 取最后一个 → 3
len(lst)                         # 长度 → 3
"|".join(["a", "b"])             # 拼接 → "a|b"
for item in lst:                 # 遍历
    print(item)
```

### 字典

```python
d = {"name": "北京", "temp": 28}

d["name"]                        # 取值 → "北京"
d.get("name")                    # 安全取值 → "北京"
d.get("xxx", "默认")              # 安全取值 + 默认 → "默认"
d["new_key"] = "value"           # 添加/修改
list(d.keys())                   # 所有键 → ["name", "temp"]
list(d.values())                 # 所有值 → ["北京", 28]
for k, v in d.items():           # 遍历键值对
    print(k, v)
```

## 要点

> - `dict.get(key)` 比 `dict[key]` 更安全 — 宁可返回 None 也不要报错
> - `for k, v in dict.items()` 是遍历字典的标准写法
> - 用字典存储函数 → `{"名称": 函数}` → 按名称调用，可以替代大量的 `if/elif`
> - `"\n".join(list)` 把列表变字符串，`str.split("\n")` 把字符串变列表，是一对互逆操作
