# 🐍 05 — Python 包与模块导入

## 什么是包 (Package)？

包就是一个**包含 `__init__.py` 文件的文件夹**，里面可以放多个 `.py` 文件（模块）。

```
tools/                   ← 这是一个"包"
├── __init__.py          ← 告诉 Python 这是一个包
├── get_weather.py       ← 这是一个"模块"
└── get_attraction.py    ← 另一个模块
```

## 本项目的包结构

```
simple_agent/
├── main.py              ← 入口
├── tools/               ← "工具" 包
│   ├── __init__.py      ← 注册和导出工具函数
│   ├── get_weather.py
│   └── get_attraction.py
└── agent_core/          ← "Agent 核心" 包
    ├── __init__.py
    ├── llm_client.py
    ├── parser.py
    └── agent.py
```

## 导入方式

### 方式 1：从 `tools` 包导入

```python
# tools/__init__.py — 在 __init__.py 中注册
from .get_weather import get_weather
from .get_attraction import get_attraction

available_tools = {
    "get_weather": get_weather,
    "get_attraction": get_attraction,
}
```

```python
# agent.py — 直接使用
from tools import available_tools

result = available_tools["get_weather"](**kwargs)
```

### 方式 2：直接从模块导入

```python
from tools.get_weather import get_weather
from agent_core.llm_client import OpenAICompatibleClient
```

### 方式 3：相对导入（包内部使用）

```python
# tools/__init__.py 中
from .get_weather import get_weather  # "." 表示"当前包"
from .get_attraction import get_attraction

# agent_core/agent.py 中
from .llm_client import OpenAICompatibleClient  # 导入同包内的模块
from .parser import parse_actions
```

## 各种导入写法

| 写法 | 含义 |
|------|------|
| `import os` | 导入标准库 |
| `import requests` | 导入第三方库 |
| `from tools import get_weather` | 从 tools 包导入 get_weather 模块 |
| `from tools.get_weather import get_weather` | 从 tools.get_weather 模块导入 get_weather 函数 |
| `from .llm_client import OpenAICompatibleClient` | 从"当前包的 llm_client 模块"导入类（相对导入） |
| `from config import API_KEY` | 从同目录的 config.py 导入变量 |

## `__init__.py` 的作用

```python
# tools/__init__.py
# 1. 让 Python 把 tools/ 当作包
# 2. 在这里"注册"包对外暴露的内容

from .get_weather import get_weather
from .get_attraction import get_attraction

# 这样外部只需要: from tools import get_weather
# 而不需要: from tools.get_weather import get_weather
```

## 要点

> - 只要文件夹里有 `__init__.py`，它就是 Python 包
> - `__init__.py` 可以**为空文件**，但必须有
> - 在包内部用 **相对导入** `from .xxx import yyy`
> - 在入口文件 (`main.py`) 用 **绝对导入** `from tools import xxx`
