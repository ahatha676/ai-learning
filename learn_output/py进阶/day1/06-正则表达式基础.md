# 🐍 06 — 正则表达式基础 (re 模块)

## 为什么需要正则？

当 LLM 返回一段文本，我们需要从中**提取特定格式的信息**，比如 `Action: get_weather(city="北京")`。用普通字符串操作很麻烦，正则表达式是最佳工具。

## 本项目用到的地方

```python
import re

# parser.py 中的 3 个关键正则操作

# 1️⃣ 提取 "Action: xxx" 这一行
action_match = re.search(r"Action: (.*)", llm_output, re.DOTALL)

# 2️⃣ 提取函数名: get_weather(...) 中的 get_weather
tool_name_match = re.search(r"(\w+)\(", action_str)

# 3️⃣ 从括号里提取键值对: arg1="val1", arg2="val2"
kwargs = dict(re.findall(r'(\w+)="([^"]*)"', args_str))
```

## 正则基础语法

| 写法 | 含义 |
|------|------|
| `r"..."` | **原始字符串**，告诉 Python 不要把 `\` 当作转义符 |
| `.*` | 匹配任意字符 0 次或多次 |
| `\w+` | 匹配一个或多个"单词字符"（字母、数字、下划线） |
| `(.*)` | **捕获组** — 把匹配到的内容提取出来 |
| `[^"]*` | 匹配除 `"` 以外的任意字符 0 次或多次 |
| `re.DOTALL` | 让 `.` 也能匹配换行符 |

## re 模块的 3 个核心函数

```python
import re

text = "Action: get_weather(city=\"北京\")"

# 1. re.search() — 查找第一个匹配
match = re.search(r"Action: (.*)", text)
if match:
    print(match.group(1))  # 输出: get_weather(city="北京")

# 2. re.findall() — 查找所有匹配
pairs = re.findall(r'(\w+)="([^"]*)"', text)
print(pairs)  # 输出: [('city', '北京')]

# 3. re.match() — 从开头匹配（类似 search，但必须从开头）
match = re.match(r"Action", text)  # 匹配成功
```

## 逐步拆解复杂正则

```python
# 完整正则:
r'(Thought:.*?Action:.*?)(?=\n\s*(?:Thought:|Action:|Observation:)|\Z)'

# 拆解:
# (Thought:.*?Action:.*?)  ← 捕获组：匹配 "Thought:" 到 "Action:" 之间的内容
# (?=...)                  ← 前瞻断言：匹配到后面出现以下内容为止
# \n\s*(?:Thought:|...)   ← 换行后出现 Thought/Action/Observation
# |                        ← 或者
# \Z                       ← 字符串结尾
```

## 本项目解析 Action 的完整流程

```python
# LLM 可能输出:
# Thought: 用户需要查询北京的天气，我先调用天气工具。
# Action: get_weather(city="北京")

# 解析步骤:
# 1. re.search(r"Action: (.*)", output) → 提取 "get_weather(city="北京")"
# 2. re.search(r"(\w+)\(", action_str)  → 提取函数名 "get_weather"
# 3. re.findall(r'(\w+)="([^"]*)"', ...) → 提取参数 {"city": "北京"}
# 4. available_tools["get_weather"](city="北京") → 调用工具
```

## 要点

> - 写正则时**用原始字符串 `r"..."`**，否则要写很多 `\\`
> - 先在小工具里测试正则，再放进代码（可以用 [regex101.com](https://regex101.com) 在线测试）
> - `(组)` 提取内容，`(?:组)` 只匹配不提取
> - 复杂的正则分步写，不要试图一行写完
